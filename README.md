# patreon-dl

A Patreon downloader written in [Node.js](https://nodejs.org).

This repo contains the `patreon-dl` library and its command-line tool. For GUI application, check out [patreon-dl-gui](https://github.com/patrickkfkan/patreon-dl-gui).

> [!NOTE]
> This is a fork I created because the original [patreon-dl](https://github.com/patrickkfkan/patreon-dl) did not have a way to filter posts by their names, so I added it myself.
> 
> For documentation and stuff, please refer to the original repo at [patreon-dl](https://github.com/patrickkfkan/patreon-dl).
